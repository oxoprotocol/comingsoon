# OXO Protocol
A Layer-1.5 protocol to transform Bitcoin from digital gold into a DeFi and GameFi powerhouse with ZK rollups.

## Structure
- `/contracts`: Solidity smart contracts (e.g., oxo.sol, counter.sol deployed on Sepolia).
- `/docs`: Whitepaper and documentation.
- `/frontend`: Web interface (waitlist, dashboard).
- `/zk`: Zero-knowledge proof circuits.

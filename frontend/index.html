Overview

What is OXO Protocol?

Layer-1.5 Blockchain

OXO Protocol is a Layer-1.5 blockchain that extends Bitcoin's trust-minimized security into a scalable, modular ecosystem.

High-Performance Apps

With BFT consensus, ZK-rollups, and native Bitcoin settlement, OXO powers high-performance applications from gaming and tokenized markets to programmable liquidity.

Unified Ecosystem

Bridge, swap, stake, and roll-up — all within one cohesive protocol. OXO unifies Bitcoin liquidity with Ethereum DeFi in a single, seamless experience.

Native Yield Staking

OXO Protocol staking allows users to secure the network while earning native yield from both OXO tokens and bridged assets like Bitcoin. Stake oxoBTC to earn OXO emissions, or stake OXO to earn protocol fees in ETH + oxoBTC.
